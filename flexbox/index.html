Flexbox

Flexbox is a CSS layout module that makes it easier to design flexible responsive layout structures without using float or positioning.

Basic Concepts

  • Flex Container: The parent element that holds the flex items. It is defined by setting the display property to flex or inline-flex.
  • Flex Items: The child elements inside a flex container.
  • Main Axis: The primary axis along which flex items are laid out. It can be horizontal or vertical depending on the flex-direction property.
  • Cross Axis: The axis perpendicular to the main axis.

Common Flexbox Properties

For Flex Container:

  • display: Defines a flex container (flex or inline-flex).
  • flex-direction: Specifies the direction of the main axis (row, row-reverse, column, column-reverse).
  • justify-content: Aligns flex items along the main axis (flex-start, flex-end, center, space-between, space-around).
  • align-items: Aligns flex items along the cross axis (flex-start, flex-end, center, stretch, baseline).
  • align-content: Aligns two or more lines along the cross axis (flex-start, flex-end, center, space-between, space-around).
  • flex-wrap: Controls whether flex items wrap onto multiple lines (nowrap, wrap, wrap-reverse).

For Flex Items:

  • flex-grow: Defines the ability for a flex item to grow if necessary.
  • flex-shrink: Defines the ability for a flex item to shrink if necessary.
  • flex-basis: Specifies the initial size of a flex item before any growing or shrinking.
  • align-self: Allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
